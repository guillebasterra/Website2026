# Personal Website Project - Rough Draft

## Core Goals
- A personal portfolio/brand website.
- Aesthetic inspiration: https://shud.in/ (Minimalist, vertical menu bar, simple text, blur transitions).
- Performance: Must be "snappy" (SPA feel, no page reload delays).
- Mobile responsive: Critical priority.

2. **Projects/Experiences**: Portfolio showcase.

- **Vertical Menu Bar**: Similar to shud.in.
- **Socials**: Icons (X, YouTube, GitHub, LinkedIn) placed at the bottom of the menu bar (Reference: richardpilnick.com).

## Visual/Creative Elements
- **Blur Transitions**: Smooth transitioning between pages.
- **Dithered Art**: A fixed overlay png at the bottom of the screen that does not move when scrolling. I can provide a file for this.
